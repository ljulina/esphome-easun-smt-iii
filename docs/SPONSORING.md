# Sponsoring

This project is an independent community effort. If you find it useful, you can support its continued development and documentation.

The GitHub **Sponsor** button on the repository page links to the options configured in [`.github/FUNDING.yml`](../.github/FUNDING.yml):

- [GitHub Sponsors](https://github.com/sponsors/ljulina)
- [Ko-fi](https://ko-fi.com/ljulina)
- [Buy Me a Coffee](https://buymeacoffee.com/ljulina)
- [PayPal](https://www.paypal.com/paypalme/ljulinacz)

Sponsorship is voluntary and does not affect access to the software, documentation, or support.
